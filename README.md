# CyberChef PGP

## Objetivo

Demonstrar a geração de chaves PGP, criptografia e descriptografia de mensagens utilizando o CyberChef.

## Ferramenta

- CyberChef
- PGP (Pretty Good Privacy)

## Etapas realizadas

1. Geração do par de chaves pública e privada.
2. Criptografia de uma mensagem.
3. Salvamento das chaves.
4. Descriptografia da mensagem utilizando a chave privada.

## Aprendizado
A atividade demonstrou como a criptografia assimétrica protege informações utilizando pares de chaves, garantindo confidencialidade e segurança dos dados.
